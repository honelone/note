Vue3中的指令 ​

01. v-model ​

（1）默认名称改变 ​

  • 在vue3中，v-model默认绑定的名称改变

    • prop：value ==> modelValue
    • 事件：input ==> update:modelValue

  • 自定义组件上的 v-model 相当于传递了 modelValue prop并接收抛出的 update:modelValue 事件

    vue
    <ChildComponent v-model="pageTitle" />

<!-- 是以下的简写: -->

<ChildComponent
  :modelValue="pageTitle"
  @update:modelValue="pageTitle = $event"
/>

（4）修饰符 ​

  • .lazy：会将绑定的事件，由input切换为change

  • .number：会将默认绑定的值，由string类型切换为number类型

  • .trim：会自动过滤掉输入的空格值

    注意：

    • 如果属性值和修饰符搭配使用，修饰符需要在属性值后面，如v-model:value.number

02. v-on ​

（1）基本概念 ​

  • 可以监听普通元素的原生DOM事件，或者监听子组件emits派发的自定义事件

  • 使用示例：

    vue
    <button @click="handleClick"></button>

<ChildComponent @customEvent="handleCustomEvent" @update="handleUpdate('msg', $event)" />

<script setup>
const handleClick = (e) => {
    console.log(e) // Event
}

const handleCustomEvent = (params) => {
    console.log(params) // 可以接收到 子组件派发的参数
}

const handleUpdate = (msg, e) => {
    console.log(msg) // 父组件上传递的参数
    console.log(e) // 子组件派发出来 参数
    
}
</script>

03. v-bind ​

（1）基本概念 ​

  • 用于动态绑定属性attribute，子组件可以通过props来接收这个属性

  • 使用示例

    vue
    <!-- 1. 普通绑定方式 -->
<img :src="imgSrc" />

<!-- 2. 动态属性名 -->
<button :[key]="value"></button>

<!-- 3. 将当前 props 中的属性传递给子组件 -->
<MyComponent v-bind="$props" />

<!-- 4. 将当前 attrs 中的属性传递给子组件 -->
<MyComponent v-bind="$attrs" />

04. v-slot ​

（1）基本概念 ​

  • 通过<slot>元素定义一个插槽出口，由父元素传入的插槽内容，将会显示在<slot>所在的位置

  • 使用示例

    vue
    <!-- 子组件中 -->
<button>
    <slot></slot>
</button>

<!-- 父组件中 -->
<ChildComponent>
    Click!!!
</ChildComponent>

<!-- 最终渲染 -->
<button>
    Click!!!
</button>

（2）进阶 ​

  • 具名插槽：<slot>元素可以通过name属性来为插槽命名，在父组件中则可以通过v-slot来指定使用哪一个插槽

    • 默认命名为default，可以不写
    vue
    <!-- 子组件中 -->
<button>
    <slot></slot>
    <!-- <slot name="default"></slot> -->
    <slot name="header"></slot>
</button>

<!-- 父组件中 -->
<ChildComponent>
    
	渲染到默认插槽
    
    <template #default>
		渲染到默认插槽
    </template>
    
	<template #header>
    	渲染到 name 为 header 的插槽
    </template>
</ChildComponent>
