01. 渲染函数 ​

  • h()函数和createNode()函数

    • 实际上，这两者的效果是等同的，h()只是createNdoe()的简写方式

（1）基本结构 ​

  • h(type, props, children)

  • type：

    • 必填项
    • 当前创建的节点类型
    • 是一个字符串，或者引入的其它组件

  • props：

    • 选填项，可省略
    • 当前节点上的属性
    • 是一个对象

  • children

    • 选填项
    • 当前节点的子节点
    • 可以是字符串，表示InnerHtml；可以是数组，表示内部节点
    jsx
    import { h } from 'vue';

// <div id="vnode1" class="vnode"> vnode1 </div>
const vnode1 = h(
    'div',
    {
        id: 'vnode1',
        class: 'vnode'
    },
    'vnode1'
)

// <div id="vnode2" class="vnode"> <span> vnode2 </span> </div>
const vnode2 = h(
    'div',
    {
        id: 'vnode2',
        class: 'vnode'
    },
    [
        h(
            'span',
            // props 可省略，可直接写 children 
            'vnode2'
        )
    ]
)

03. 简单使用 ​

tsx
import { defineComponent } from 'vue';

# render
export default defineComponent({
    props: {},
    render(props) {
        ...
        return <div>JSX - render</div>
    }
})

# setup
export defaultComponent({
    props: {},
    setup(props) {
        ...
        // 注意这里是以函数的方式返回
        return () => {
            return <div>JSX - setup</div>
        }
    }
})
