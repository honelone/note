函数 ​

01. 定义 ​

  • 函数类型：定义函数类型时，需要为形参添加类型，需要为函数返回值添加类型

    typescript
    // 命名函数
function fun1(x: number, y: number): number {
  return x + y
}

// 匿名函数
let fun2 = function(x: number, y: number): number { 
  return x + y
}

// 箭头函数:
let fun3(x: number, y: string) => string


// 一个完整的函数类型
let myAdd: (x: number, y: number) => number = 
    function(x: number, y: number): number => {
        return x + y
    }

// 表示一个函数类型, 参数1是 number 类型， 参数2是 number 类型，返回值时 number 类型
// 如果这里不写类型，则表示它是一个 any 类型
type AddFnType = (num1: number, num2: number) => number

const add: AddFnType = (a1: number, a2: number) => {
    return a1 + a2
}

02. 参数 ​

（1）可选参数 ​

  • 可选参数：可以在参数名后面添加?标识符来标识这个参数是可选的

    • 可选类型可以看作是：指定的类型和undefined的联合类型，即不传则为undefined类型
    • 可选参数要写在必选类型的后面
    typescript
    // name 参数可选,非必传
function func(name?: string): void {
	// do something
}

（2）默认参数 ​

  • 默认参数：可以直接为参数赋值，让其有一个默认值

    • 默认参数也可以看作是指定类型和undefined的联合类型
    • 如果默认参数在第一个，则可以传undeinfed来表示使用默认值
    typescript
    // name 参数设置默认值
function func(name: string = 'Ruovan'): void {
	// do something
}

（3）剩余参数 ​

  • 剩余参数：可以通过省略号...来接收剩余参数

    • 表示没有接收的参数数组，或者不定参数
    • 剩余参数的参数名是...后面的名字，可以指定，它表示一个数组，可以在函数内使用这个数组
    typescript
    // 剩余参数 ... 
function func(name: string, ...args: string[]):void {
    console.log(name, args)
}

03. 函数重载 ​

  • 函数重载是指：函数名相同，而形参不同的多个函数

    • 声明函数时，参数类型要具体
    • 实现函数时，参数类型要宽泛
    • 使用函数时，参数类型要与声明函数时的一致
    typescript
    /* 
    函数重载: 函数名相同, 而形参不同的多个函数
    需求: 我们有一个add函数，它可以接收2个string类型的参数进行拼接，也可以接收2个number类型的参数进行相加 
    */

// 重载函数声明
function add (x: string, y: string): string
function add (x: number, y: number): number

// 定义函数实现
function add(x: any, y: any): any {
    rturn x + y
}

console.log(add(1, 2))
console.log(add('a', 'b'))
// console.log(add(1, 'a')) // error

  • 作用：以上面例子

    • 如果在一个函数里面，通过联合类型来对参数进行类型注解
    • 然后在函数体通过typeof来进行类型判断
    • 那么会导致函数体变大，需要作许多逻辑判断，
    • 且无法确定函数返回值的类型
    typescript
    function add(x: number | string, y: number | string): number | string {
    if(typeof x == 'number' && typeof y == 'number'){
        return x + y
    } else if(typeof x == 'string' && typeof y == 'string'){
        return x + y
    } else {
        return ''
    }
}
