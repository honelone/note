一、基础知识 ​

01. 基本概念 ​

  • HTML：Hyper Text Markup Language，超文本标记语言，它不是一种编程语言，而是一种标记语言

02. 基本结构 ​

  • 这里仅作HTML5的结构的描述

    html
    <!DOCTYPE html>
<!--文档声明：位于文档的首行，告诉浏览器文档使用哪个HTML版本。-->

<html>
  <!-- html标签：告诉页面这是一个HTML文档，所有网页内容包含在html标签内 -->

  <head>
    <!--head标签:就是一个网页的头,头部中的内容一般存放描述文档的信息。-->

    <!--meta元素可提供有关页面的元信息-->
    <!--比如针对搜索引擎和更新频度的描述和关键词等。-->
    <meta charset="UTF-8" />
    <!--声明文件的编码是UTF-8的编码，告诉浏览器，你用什么编码去解析这个文件-->

    <meta name="keywords" content="Java培训、web前端、UI培训" />
    <meta name="description" content="源代码教育品牌升级为源码时代" />
    <!-- 搜索引擎优化SEO，提供给搜索引擎的关键字信息 -->

    <title>这是标题</title>
    <!--title : 网页的标题，会显示在标题栏-->
  </head>

  <body>
    <!--这里存放网页主体内容-->
  </body>
</html>

二、基础标签 ​

01. 文字和段落标签 ​

  • <h>标题标签：用于设置标题

    • 可按照数字分为1 级 ~ 6 级标题，分别为<h1>、<h2>、...、<h6>

  • <p>段落标签：用于设置段落文字

    • 每一个段落跟另外的内容间距会很明显，这是因为浏览器会默认设置 p 标签的内边距和外边距

  • <br>换行标签：用于设置换行

    • 换行相对于段落而言，间距会小很多

  • <hr>水平线标签：用于表现分割线

  • <b>加粗标签和<strong>强调标签

    • <b>标签是单纯的加粗，只有视觉上的效果
    • <strong>标签除了视觉上的加粗之外，还具有强调的实际语义

  • <i>斜体标签和<em>斜体强调标签

    • <i>只是单纯的斜体，没有任何语义
    • <em>除了斜体以外，还有强调的语义

  • <small>小号字标签和<big>大号字标签

    • <small>标签，主要用于小号字，没有语义
    • <big>标签只有字体修饰功能，没有语义

  • <del>删除标签和<ins>插入标签

    • <del>标签用于设置删除线
    • <ins>标签用于设置下划线

  • <sub>下标标签和<sup>上标标签

    • <sub>标签用于设置下标显示
    • <sup>标签用于设置上标显示

02. 特殊符号 ​

  • 以&开头，以;结束

    字符标识含义
    &nbsp;空格（小空格）
    &emsp;空格（大空格，一个中文字符）
    &gt;大于号：>
    &lt;小于号：<
    &copy;版本号：©

03. 布局标签 ​

  • 块元素：<div>

    • 块元素独占一行，宽度默认为外界的 100%，可设置高度、宽度

  • 行内元素：<span>

    • 行内元素不独占一行，宽度默认为内部内容的宽度，无法设置高度、宽度

04. 超链接标签<a> ​

  • 基本属性

    属性名描述
    hrefhyper reference，用于设置目标链接地址
    target用于设置窗口跳转，可设置为 _self（默认值）表示在当前窗口打开链接，设置为 _blank表示在新窗口打开链接
    name可用于创建锚点，用name创建的锚点，只能从一个a标签跳转到另一个a标签，即只能在a标签之间进行跳转
    id创建锚点的另一种方式，可以在不同标签之间跳转

  • 锚点示例

    html
    <!-- 定义锚点 -->
<a name="top">这是顶部</a>

<div style="height:3000px;"></div>

<!-- 定义跳转链接 -->
<a href="#top">点击跳转回到顶部</a>

<!-- ------------分割线------------ -->

<!-- 定义锚点 -->
<div id="top">这是顶部</div>

<div style="height:3000px;"></div>

<!-- 定义跳转链接 -->
<a href="#top">点击跳转回到顶部</a>

（3）单元格合并属性 ​

属性名作用

  </tr>
  <tr>
    <!-- <td>第三行单元格1消失了，所以不需要再写</td> -->
    <td>第三行单元格2</td>
  </tr>
  <tr>
    <td>第四行单元格1</td>
    <td>第四行单元格2</td>

（3）表格样式的说明 ​

  • 边框：使用border-collapse属性取消表格间隙

    • 当给表格添加边框，是在最外层添加一个边框
    • 给单元格添加边框是在每一个单元格元素上添加边框
    • 当两者同时设置，就会出现边框间隙
    • 可设置border-collapse: collapse;用于取消这个间隙
      • 只有在设置了border-collapse属性
      • 对thead、tbody、tr 设置边框的操作才会生效

  • 大小：可以通过width 属性和height 属性来设置表格的宽高

    • 但不能小于表格的最小宽高

      • 如果小于了单元格内容的宽度

      • 那么单元格里面的内容会换行显示，导致高度增加

      • 换行显示最小为每行一个字

      • 所以，如果小于了表格的最小高度，该项设置无效

    • 默认这个最小宽高是表格里面内容大小

    • 一般通过设置单元格的宽高来撑开表格的大小

    • 每一列的单元格（th、td）的宽度都是一样的，以最大的为准

（2）<input> 标签 ​

  • input 的 type 属性

    属性值input 形态特点
    text文本框默认形态，收集文本信息
    password密码框隐藏输入内容
    submit提交按钮点击后执行提交
    button普通按钮可结合 JS 脚本实现自定义功能
    reset重置按钮回到最初状态（注意不是清空）
    radio单选一组数据，只选择一项
    （name 必须设置一样的名字，且提供 value 值）
    checkbox多选一组数组，可以选择多项
    （name 必须设置一样的名字，且提供 value 值）
    file文件上传可以选择文件以提交
    hidden隐藏域不会展示到页面中，但会提交
    image图像域

  • <input>的其它属性

    属性值描述
    name定义 input 收集到的数据名称
    只能使用数字、字母、下划线命名，且不能以数字开头
    value用于存放收集到的数据
    文本框和密码框的 value 可以不用写，设置了即默认值
    其它无法由用户输入的 input ，需要在标签中设置 value 值
    disabled定义标签不可用
    checked适用于 radio、checkbox，用于设置默认选中项
    maxlength规定用户输入的最大字符长度
    size规定文本框的宽度
    placeholder设置输入框的输入提示
    autocomplete基于之前输入的值，提示写过的内容，自动输入

三、HTML5 标签 ​

01. 布局标签 ​

标签描述意义
<header>头部标签，定义了文档的头部区域
<nav>导航栏标签，定义了导航链接
<main>定义页面主要内容,它不应包含在文档中重复出现的内容，比如侧栏、导航栏、版权信息、站点标志或搜索表单。
<section>定义某个区域，定义文档中的节（section、片段、板块）。
<article>内容标签，定义页面独立的内容区域。
<aside>侧边栏标签，定义页面的侧边栏内容。
<footer>底部标签，定义 section 或 document 的页脚。
