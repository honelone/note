数组 ​

列表 ​

  • 一组有序的数据，其列表元素可以是任意类型，可以是任意长度，但受内存限制

  • listSize：表示列表长度

  • append()：表示在列表末尾新增一个元素

  • insert()：表示在给定元素之后，或列表起始位置，插入一个元素

  • remove：表示删除一个元素

  • clear：表示清空列表中所有元素

  • toString：显示列表中所有的元素，以字符串返回

  • getElement()：显示当前元素

  • next()：表示从当前元素移动到下一个元素

  • prev()：表示从当前元素移动到前一个元素

  • moveTo(n)：表示直接移动到指定位置，n表示位置，currPos属性表示列表当前位置

    属性/方法描述
    listSize列表的元素个数

栈 ​

  • 栈是一种特殊的列表，
    • 不同的是，栈内的元素只能通过列表的一端访问， 这一端称为栈顶
    • 栈是一种后入先出（LIFO， last-in-first-out） 的数据结构
  • 入栈使用 push() 方法， 出栈使用 pop() 方法

队列 ​

  • 队列也是一种特殊的列表
    • 不同的是，队列只能在队尾插入元素， 在队首删除元素
    • 队列是一种先进先出（FIFO，First-In-First-Out） 的数据结构
  • 入队操作：在队尾插入push()新元素， 出队操作：在队头删除shift()元素

链表 ​

  • JS 中，数组也是对象，可以使用链表来代替

  • 链表是，由一组节点组成的集合。 每个节点都使用一个对象的引用指向它的后继

    • 指向另一个节点的引用叫做链

    • 链表最前面有一个特殊节点， 叫做头节点header

    • 链表的尾元素指向一个 null 节点

      js
      header -> A -> B -> null
      1

  • 链表元素则是靠相互之间的关系进行引用

  • 向链表中插入一个节点

    • 需要修改它前面的节点（前驱）， 使其指向新加入的节点

    • 而新加入的节点则指向原来前驱指向的节点

      js
      A -> B
A -> C && C -> B
      1
      2

  • 从链表中删除一个元素

    • 将待删除元素的前驱节点指向待删除元素的后继节点

    • 同时将待删除元素指向 null， 元素就删除成功了

      A -> B -> C
A -> C && B -> null

// Node 类， element用来保存节点上的数据， next 用来保存指向下一个节点的链接
function Node(element) {
  this.element = element;
  this.next = null;
}

// LinkedList类， 提供了对链表的操作方法
function LList() {
  this.head = new Node('head');
  this.find = find;
  this.insert = insert;
  this.remove = remove;
  this.display = display;
}
// find 方法，用于查找指定数据，并返回数据节点的链接
function find(item) {
  var currNode = this.head;
  while (currNode.element != item) {
    currNode = currNode.next;
  }
  return currNode;
}
// insert 方法，用于插入节点 在item节点后插入新节点newElement
function insert(newElement, item) {
  var newNode = new Node(newElement);
  var current = this.find(item); // 找到指定的 Item元素
  newNode.next = current.next; // 把指定的 item 元素的 next 指向赋值给新的 newElement
  current.next = newNode; // 然后让 item 元素的 next 指向 新的 newElement
}
// findPrevious 方法， 用于 查找指定数据的 前一个节点
function findPrevious(item) {
  var currNode = this.head;
  while (!(currNode.next == null) && currNode.next.element != item) {
    currNode = currNode.next;
  }
  return currNode;
}
// remove 方法，用于删除指定节点
function remove(item) {
  var prevNode = this.findPrevious(item); // 先找到要删除节点的 前一个节点
  if (!(prevNode.next == null)) {
    prevNode.next = prevNode.next.next; // 让前一个节点指向 其子节点（删除节点） 的后一个节点
  }
}
// display 方法， 用于显示链表中的元素
function display() {
  var currNode = this.head;
  while (!(currNode.next == null)) {
    print(currNode.next.element);
    currNode = currNode.next;
  }
}

双向链表 ​

  • 给 Node 对象增加一个属性， 该属性存储指向前驱节点的链接

  • 删除节点时， 效率提高了， 不需要再查找待删除节点的前驱节点了

    js
     A <-> B <->C
    1
    js
    // Node 类， element用来保存节点上的数据
// next 用来保存指向下一个节点的链接
// previous 用来保存指向上一个节点的链接
function Node(element) {
  this.element = element;
  this.next = null;
  this.previous = null;
}
// insert 方法，需要再增加一个指向
function insert(newElement, item) {
  var newNode = new Node(newElement);
  var current = this.find(item);
  newNode.next = current.next;
  newNode.previous = current; // 增加的
  current.next = newNode;
}

// remove 方法，不需要再查找前驱节点了
// 同样需要新增
function remove(item) {
  var currNode = this.find(item);
  if (!(currNode.next == null)) {
    currNode.previous.next = currNode.next;
    currNode.next.previous = currNode.previous; // 新增的
    currNode.next = null;
    currNode.previous = null; // 新增的
  }
}

// findLast 方法，进行逆序查找
function findLast() {
  var currNode = this.head;
  while (!(currNode.next == null)) {
    currNode = currNode.next;
  }
  return currNode;
}

循环链表 ​

  • 在创建循环链表时， 让其头节点的 next 属性指向它本身：head.next = head
  • 这种行为会传导至链表中的每个节点， 使得每个节点的 next 属性都指向链表的头节点
    • 换句话说， 链表的尾节点指向头节点， 形成了一个循环链表

二叉树 ​

  • 一棵树最上面的节点称为根节点

    • 如果一个节点下面连接多个节点， 那么该节点称为父节点
    • 它下面的节点称为子节点
    • 一个节点可以有 0 个、 1 个或多个子节点
    • 没有任何子节点的节点称为叶子节点

  • 二叉树是一种特殊的树， 它的子节点个数不超过两个

    • 每个节点都有一个与之相关的值， 该值有时被称为键
    • 一个父节点的两个子节点分别称为左节点和右节点

  • 沿着一组特定的边， 可以从一个节点走到另外一个与它不直接相连的节点。 从一个节点到另一个节点的这一组边称为路径

  • 以某种特定顺序访问树中所有的节点称为树的遍历

  • 树可以分为几个层次， 根节点是第 0 层， 它的子节点是第 1 层， 子节点的子节点是第 2 层， 以此类推

    • 树中任何一层的节点可以都看做是子树的根， 该子树包含根节点的子节点， 子节点的子节点等。
    • 我们定义树的层数就是树的深度

  • 二叉树的深度为：根节点到最远叶子节点的最长路径上的节点数。

二叉查找树（BST） ​

  • 二叉查找树是一种特殊的二叉树， 相对较小的值保存在左节点中， 较大的值保存在右节点中
  • 有一个 insert() 方法， 用来向树中加入新节点
    • (1) 设根节点为当前节点。
    • (2) 如果待插入节点保存的数据小于当前节点， 则设新的当前节点为原节点的左节点； 反之， 执行第 4 步。
    • (3) 如果当前节点的左节点为 null， 就将新的节点插入这个位置， 退出循环； 反之， 继续执行下一次循环。
    • (4) 设新的当前节点为原节点的右节点。
    • (5) 如果当前节点的右节点为 null， 就将新的节点插入这个位置， 退出循环； 反之， 继续执行下一次循环。
  • 有三种遍历 BST 的方式： 中序、 先序和后序
    • 中序遍历：按照节点上的键值， 以升序访问 BST 上的所有节点。
    • 先序遍历：先访问根节点， 然后以同样方式访问左子树和右子树。
    • 后序遍历：先访问叶子节点， 从左子树到右子树， 再到根节点
  • 有三种类型的 BST 查找
    • 查找给定值：比较该值和当前节点上的值的大小
    • 查找最小值：遍历左子树
    • 查找最大值：遍历右子树

搜索 ​

  • 深度优先搜索（DFS）
    • 从一条路径的起始顶点开始追溯， 直到到达最后一个顶点
      • 然后回溯，继续追溯下一条路径， 直到到达最后的顶点
      • 如此往复， 直到没有路径为止
    • 通过搜索来查看在图中有哪些路径可以选择
    • 思路：
      • 访问一个没有访问过的顶点， 将它标记为已访问，
        • 再递归地去访问在初始顶点的邻接表中其他没有访问过的顶点
  • 广度优先搜索
    • 从第一个顶点开始， 尝试访问尽可能靠近它的顶点。
    • 本质上， 这种搜索在图上是逐层移动的，
    • 首先检查最靠近第一个顶点的层， 再逐渐向下移动到离起始顶点最远的层
    • 思路
      • (1) 查找与当前顶点相邻的未访问顶点， 将其添加到已访问顶点列表及队列中；
      • (2) 从图中取出下一个顶点 v， 添加到已访问的顶点列表；
      • (3) 将所有与 v 相邻的未访问顶点添加到队列。

基本排序 ​

  • 冒泡排序

    • 循环比较两两相邻的数据， 当左侧值大于右侧值时将它们进行互换
      • 每次只能将一个数确定位置
    js
    function bubbleSort(arr) {
  let len = arr.length;
  for (let i = 0; i < len; i++) {
    for (let j = 0; j < len - i; j++) {
      if (arr[j] > arr[j + 1]) {
        let temp = arr[j + 1];
        arr[j + 1] = arr[j];
        arr[j] = temp;
      }
    }
  }
}

  • 选择排序

    • 每一次从待排序的数组元素中选择最大(最小)的一个元素作为首元素,直到排完为止
    • 选择排序从数组的开头开始， 将第一个元素依次和其他元素进行比较
    • 检查完所有元素后， 会得到当前循环中最小的元素的下标，然后进行交换，最小的元素会被放到数组的第一个位置
    • 然后算法会从第二个位置继续
    js
    function selectionSort(arr) {
  let min = 0;
  for (let i = 0; i < arr.length - 1; i++) {
    min = i;
    // 寻找最小值的下标
    for (var j = i + 1; j < arr.length; j++) {
      if (arr[j] < arr[min]) {
        min = j;
      }
    }
    // 交换
    if (min !== i) {
      [arr[i], arr[min]] = [arr[min], arr[i]];
    }
  }
}

  • 插入排序

    • 通过构建有序序列，对于未排序数据，在已排序序列中从后向前扫描，找到相应位置并插入
    • 插入排序有两个循环
    • 外循环将数组元素挨个移动， 而内循环则对外循环中选中的元素及它后面的那个元素进行比较。 如果外循环中选中的元素比内循环中选中的元素小，那么数组元素会向右移动， 为内循环中的这个元素腾出位置
    js
    function insertionSort(arr) {
  for (let i = 0; i < arr.length; i++) {
    let preIndex = i - 1;
    let current = arr[i];
    // 当 preIndex 大于0时，并且当前循环到的值比 preIndex 指向的值小，进入循环内部
    while (preIndex >= 0 && current < arr[preIndex]) {
      // 如果当前值比 preIndex 指向的值小，则将 preIndex 指向的值往后移一位，即让 preIndex+1 的值等于 preIndex 的值
      arr[preIndex + 1] = arr[preIndex];
      preIndex--;
    }
    // 循环结束，表示 preIndex为-1 或者 current 大于了 preIndex 指向的值
    // -- 那么 preIndex+1 的位置，即是我们要插入的位置
    arr[preIndex + 1] = current;
  }

  return arr;
}

高级排序 ​

  • 希尔排序

  • 归并排序

    • 将两个有序数列合并成一个有序数列，我们称之为归并
    • 将一个数组对半拆分成两个小数组，然后两个小数组继续拆分，直到每个小组只有一个元素
    • 然后再按照上面的拆分过程逐步合并每个小组，由最底层的一个元素合并成两个元素，合并后的每个小组都是有序的
    • 按照这种思路一直往上合并
    js
    // 用于合并两个数组
function merge(left, right) {
  let result = [];
  // 当两个数组的长度不为0时，表示两者内部都有元素
  while (left.length && right.length) {
    // 连个数组都是有序的，所以直接比较第一个元素，然后再push到result中
    if (left[0] < right[0]) {
      result.push(left.shift());
    } else {
      result.push(right.shift());
    }
  }
  // 然后上一步结束，则表示有一个数组的长度为0，但另一个数组不一定为0，所以这里要对两者都进行处理
  while (left.length) {
    result.push(left.shift());
  }
  while (right.length) {
    result.push(right.shift());
  }

  return result;
}

// 主方法
function mergeSort(arr) {
  if (arr.length <= 1) return arr;

  let mid = Math.floor(arr.length / 2);
  let left = arr.splice(0, mid);
  let right = arr.splice(mid);

  return merge(mergeSort(left), mergeSort(right));
}

  • 快速排序

    • 首先要在列表中选择一个元素作为基准值（pivot）。
      • 数据排序围绕基准值进行，将列表中小于基准值的元素移到数组的底部（左侧）， 将大于基准值的元素移到数组的顶部（右侧）
      • 然后递归处理左侧和右侧两个数组
    js
    function quickSort(arr) {
  if (arr.length <= 1) return arr;

  let mid = Math.floor(arr.length / 2);
  // 挑选中间值作为基准值
  let pivot = arr.splice(pivot, 1);
  let left = [];
  let right = [];

  for (let i = 0; i < arr.length; i++) {
    // 小于基准值的push到left中， 大于基准值的push到right中
    if (arr[i] < pivot) {
      left.push(arr[i]);
    } else {
      right.push(arr[i]);
    }
  }

  return quickSort(left).concat([pivot], quickSort(right));
}
